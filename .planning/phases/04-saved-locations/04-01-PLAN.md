**2. Create `src/hooks/useSavedLocations.ts`:**

Constants:
- `STORAGE_KEY = 'settl-saved-locations-v1'`
- `MAX_ITEMS = 10`

`load()` function:
- `localStorage.getItem(STORAGE_KEY)` wrapped in try/catch
- Parse JSON, validate `Array.isArray(parsed)`, return `SavedLocation[]`
- On any error or non-array: return `[]`

`persist(items: SavedLocation[])` function:
- `localStorage.setItem(STORAGE_KEY, JSON.stringify(items))` wrapped in try/catch
- Fail silently (handles QuotaExceededError in incognito)

`useSavedLocations()` hook — returns `{ locations, isSaved, toggle, remove }`:

- `locations`: `useState<SavedLocation[]>(() => load())` — lazy initializer
- `isSaved(lat: number, lon: number): boolean` — useCallback, computes id via `${lat.toFixed(5)},${lon.toFixed(5)}`, returns `locations.some(l => l.id === id)`
- `toggle(entry: { label: string; lat: number; lon: number }): { rejected: boolean }`:
  - Compute `id = ${entry.lat.toFixed(5)},${entry.lon.toFixed(5)}`
  - If id already exists in `locations`: remove it (filter), persist, return `{ rejected: false }`
  - If `locations.length >= MAX_ITEMS`: do NOT add, return `{ rejected: true }`
  - Otherwise: append `{ ...entry, id, savedAt: Date.now() }`, persist, return `{ rejected: false }`
  - IMPORTANT: compute `rejected` from closure state BEFORE calling `setLocations` (see Pitfall 1 in research). Use `setLocations(prev => ...)` for the actual state update but compute the return value from the pre-update snapshot.
- `remove(id: string): void` — useCallback, filter out by id, persist, setLocations

All useCallback dependencies must be correct: `isSaved` depends on `[locations]`, `toggle` depends on `[locations]`, `remove` depends on `[]`.

Import `SavedLocation` from `../../types` (not re-export — single source of truth in types/index.ts). Export the hook as named export `useSavedLocations`. Also re-export `SavedLocation` type from the hook file for convenience: `export type { SavedLocation } from '../types';`

**3. Create heart toggle handler:**

```typescript
function handleHeartToggle() {
  if (!coordsA) return;
  const label = resolvedA ?? `${coordsA.lat.toFixed(5)}, ${coordsA.lon.toFixed(5)}`;
  const result = toggle({ label, lat: coordsA.lat, lon: coordsA.lon });
  if (result.rejected) {
    setCapWarning(true);
    setTimeout(() => setCapWarning(false), 3000);
  } else {
    setCapWarning(false);
  }
}
```
