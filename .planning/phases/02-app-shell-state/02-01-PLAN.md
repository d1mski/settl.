       - Default value: `'system'`
       - `useState` initializer reads from localStorage, falls back to `'system'`

       - Second `useEffect` for `theme === 'system'`: adds `matchMedia('(prefers-color-scheme: dark)')` change listener, cleans up on unmount or when theme changes away from 'system'

    3. Create `src/contexts/FontScaleContext.tsx` implementing the FontScaleProvider per RESEARCH Pattern 2:
       - localStorage key: `'settl-font-scale'`
       - Constants: `MIN = 0.8`, `MAX = 1.4`, `STEP = 0.1`
       - `useState` initializer reads from localStorage, parses float, clamps to [MIN, MAX], falls back to `1`
       - `useEffect` on `scale` change: sets `document.documentElement.style.fontSize = \`${scale * 100}%\`` and writes to localStorage
       - `increase()`: `setScale(s => Math.min(MAX, parseFloat((s + STEP).toFixed(2))))`
       - `decrease()`: `setScale(s => Math.max(MIN, parseFloat((s - STEP).toFixed(2))))`
       - Exports: `FontScaleProvider` component, `useFontScale` hook

    4. Edit `index.html` — add a blocking inline `<script>` inside `<head>`, BEFORE the closing `</head>` tag and after the `<title>` tag. The script (per RESEARCH Pattern 3):
       ```
       (function() {
         var theme = localStorage.getItem('settl-theme') || 'system';
         var prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
         var isDark = theme === 'dark' || (theme === 'system' && prefersDark);
         if (!isDark) document.documentElement.classList.add('theme-light');
         var scale = parseFloat(localStorage.getItem('settl-font-scale') || '1');
         if (isFinite(scale) && scale !== 1) {
           document.documentElement.style.fontSize = (scale * 100) + '%';
         }
       })();
       ```

    5. Edit `src/main.tsx` — add imports for `ThemeProvider` from `'./contexts/ThemeContext'` and `FontScaleProvider` from `'./contexts/FontScaleContext'`. Wrap `<App />` with `<ThemeProvider><FontScaleProvider>...</FontScaleProvider></ThemeProvider>` inside `<React.StrictMode>`.
